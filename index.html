Защита проектов и самостоятельные работы

Каждый месяц вы создаете и защищаете проект, а также выполняете множество самостоятельных работ на дом для углубленного изучения.

Программа

Месяц 1: Введение в HTML, CSS и основы программирования

  1. Знакомство с основами HTML:
    • Теги, атрибуты, структура документа.
    • Формы, списки, таблицы.
  2. Основы CSS:
    • Селекторы, свойства, значения.
    • Позиционирование, Flexbox, Grid.
  3. Основы программирования:
    • Переменные, типы данных, операторы.
    • Условия и циклы.

Месяц 2: Продвинутый HTML, CSS и JavaScript

  1. Медиазапросы и адаптивный дизайн.
  2. Анимации и переходы в CSS.
  3. Знакомство с JavaScript:
    • Основы синтаксиса, функции.
    • Массивы и объекты.
    • Взаимодействие с DOM.

Месяц 3: Продвинутый JavaScript и работа с данными

  1. ES6+ синтаксис: стрелочные функции, деструктуризация.
  2. Асинхронное программирование (Promise, async/await).
  3. Работа с API и Fetch.

Месяц 4: Введение в React.js

  1. Основы React:
    • Компоненты, JSX.
    • Стейты и пропсы.
    • Жизненный цикл компонентов.
  2. Функциональные компоненты и хуки.

Месяц 5: Управление состоянием в React с Redux

  1. Знакомство с Redux:
    • Actions, reducers, store.
    • Интеграция Redux с React.
  2. Работа с Redux Toolkit.

Месяц 6: TypeScript и интеграция с React

  1. Введение в TypeScript:
    • Типы данных, интерфейсы.
    • Классы и модули.
  2. Интеграция TypeScript с React и Redux.

Месяц 7: Введение в Node.js

  1. Основы Node.js:
    • Работа с файловой системой (модуль fs).
    • Создание и настройка серверов.
  2. Express.js и создание REST API.

Месяц 8: Инструменты разработки и TanStack Query

  1. Работа с Git и GitHub:
    • Основы Git, коммиты, ветки.
    • Работа с удаленными репозиториями.
  2. Введение в TanStack Query для управления данными в React.

Месяц 9: Финальные проекты и закрепление знаний

  1. Заключительные проекты:
    • Создание SPA на React и Node.js.
    • Использование TypeScript и Redux в проекте.
    • Интеграция TanStack Query для работы с данными.
  2. Оптимизация и деплой проекта.

Вопросы

Есть ли какие-то требования к предварительной подготовке для участия в курсе?

Вам не понадобятся специальные навыки или опыт - мы обучим вас с нуля.

Какой ноутбук нужен?

  • Оперативная память: от 4 Гб
  • Процессор: Intel Core i3 начиная с 3 поколения. Лучше i5 8 поколения или аналоги от AMD
  • Жесткий диск: свободное место на HDD или SSD от 20 Гб

Предусмотрены ли скидки?

Да, скидки предусмотрены. Они выдаются за хорошую успеваемость ежемесячно от 10 до 30%

Могу ли я получить доступ к материалам курса после его завершения?

Да, вы имеете доступ к материалам и ресурсам после завершения, чтобы могли обращаться к ним в будущем, достаточно войти в LMS
